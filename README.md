# SaaS User Engagement & Churn Analysis Dashboard (Spotify Style)

## 📌 Project Overview
This project delivers an end-to-end data analytics solution tailored for a music streaming SaaS platform. Designed with a sleek, dark-mode Spotify aesthetic, the dashboard translates raw data into meaningful product and customer success insights, helping product managers track user engagement, optimize playlist curation, and mitigate churn risks.

## 🛠️ Tech Stack & Architecture
* **Data Warehouse & Extraction:** Microsoft SQL Server (Transact-SQL)
* **Data Modeling:** Star Schema (Fact & Dimension Tables)
* **BI & Visualization:** Microsoft Power BI Desktop
* **Design Theme:** Spotify Electro-Green & Dark UI

## 📊 Key Dashboard Features
1. **Genre Popularity Distribution:** A comprehensive bar chart ranking musical genres by their average popularity score, highlighting core engagement drivers.
2. **Audio Attributes Analytics (Energy Gauge):** A dynamic gauge metric tracking the average energy of tracks within selected filters against a fixed scale (0.00 to 1.00) to analyze listener mood.
3. **Melodic Mode Distribution (Donut Chart):** Interactive breakdown of track compositions by musical key (Major vs. Minor), displaying internal proportions dynamically based on visual filtering.
4. **Granular Exploration Tables:** Contextual deep-dives filtering specific tracks and artists based on selected attributes.

## 💡 Business Impact & Analytical Insights
* **Retention Strategy:** By identifying high-popularity genres with specific audio characteristics (e.g., high energy in Rap/Pop), product teams can design targeted push notifications and personalized marketing campaigns for users showing early signs of churn.
* **Product Optimization:** Cross-filtering interactions reveal exactly which genres an artist operates in, allowing content editors to strategically place rising tracks into core high-traffic playlists to maximize user session length.
